# ms-bff - Backend For Frontend

Servicio intermediario entre el frontend y los microservicios internos. Recibe las peticiones del cliente, llama a los servicios necesarios y devuelve una respuesta unificada.

Desarrollado con Java 17 y Spring Boot. Se levanta en el puerto 8084.

Para correr usar `docker compose up --build ms-bff` desde la raiz del proyecto.
